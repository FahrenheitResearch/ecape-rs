# Method Validation Paper

This directory contains the current methods-paper draft for `ecape-rs`:

- `main.pdf`: compiled paper
- `main.tex`: LaTeX source
- `figures/`: figure assets used by the paper

The paper documents direct compatibility testing against Python `ecape-parcel` 1.2.2, internal no-entrainment CAPE-limit tests, the 3,024-configuration event-sample sweep, the bounded 5,368-row random/stress/storm-motion sweep, runtime separation, and current limitations.

The paper is included here so users of the crate can inspect the validation basis for the implementation.
